MEOW Prompt Copy Helper

這份 HTML 專門給你一鍵複製 prompt。每個區塊都能單獨 Copy；上方也有整包複製按鈕。內容以 released code 的 faithful raw prompt 為主，另外把 repo 中的 discipline-aware system prompt 與 SFT packing 也整理進來。

PDF viewer 對剪貼簿按鈕的支援很不穩定，所以我把一鍵複製功能獨立做成這份 HTML。這樣你不用再手動框選。
重點提醒：CoT prompt 的 faithful raw 版本會把 Target Outline 一起傳入；outline generation 的 released formatter 只餵 title + ref_meta，而不是 paper narrative 中較抽象的 full metadata。
